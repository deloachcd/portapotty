# portapotty

## what is this?
bash scripting project for deploying dotfiles and installing apt
packages on a new linux workstation.

## "shit out" all deployments
```
git clone https://github.com/deloachcd/portapotty &&\
	cd portapotty &&\
	./deploy.sh &&\
	cd ..
```
